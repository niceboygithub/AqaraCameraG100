# Aqara Camera G100
The information of Aqara Camera G100

## firmwares
The stock firmwares : used to update firmware after booted

The recovery files  : used to recovery G100 if any problems
```
    Remove the power of the camera.
    Copy the uImage, rootfs00 and rootfs01 to MicroSD card in FAT32 format.
    Plug-in the MicroSD card to the camera.
    Press the reset button of the camera without release
    Plug-in the Type-C power cable to the camera
    When the LED turn to the purple, release the button.
    After the flash is completed, the camera will reboot to normal.
```

The rooted firmwares: used to update firmware for root
```
    Use the same method as the recovery files
```
